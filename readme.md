# Python_lab_3
Приложение для управления задачами (ToDo) с графическим интерфейсом, хранением данных в базе, визуализацией активности и веб-версией на Django.

## Описание
Программа представляет собой MVP системы учёта задач. Реализована настольная версия на Python и веб-версия на Django. Данные сохраняются в SQLite, для веб-версии предусмотрена поддержка Docker.

Функциональные возможности:
- таблица задач (название, описание, статус, даты);
- добавление новых задач;
- отметка выполнения / отмена выполнения;
- логирование активности пользователя;
- график активности (matplotlib);
- меню и выход из приложения;
- собственные исключения для обработки ошибок;
- модульные тесты pytest;
- веб-версия (Django) с просмотром, добавлением и изменением задач.

## Установка и запуск
### 1. Установка зависимостей
```
pip install -r requirements.txt
```

### 2. Запуск GUI-версии
```
python app/main.py
```

### 3. Запуск web-версии
```
docker-compose up -d
```

Открыть: localhost:8000

## Тестирование
Запуск тестов:
```
python -m pytest
```
Покрывается функциональность:
- добавление задач;
- переключение статуса;
- обработка исключений;
- работа репозитория SQLite.
